# Rollenspiel-App

Eine moderne Web-App für Rollenspiel-Spielleiter und Spieler.

## Features

### Spielleiter
- Übersicht über alle Spieler und ihre Charaktere
- Bilder an Spieler senden
- Zugriff auf alle Funktionen

### Spieler
- Zugriff auf eigene Charakterdaten (Werte und Gegenstände)
- Würfelproben auf eigene Werte ablegen
- Bilder vom Spielleiter empfangen
- Zugriff auf das Gruppentagebuch (lesen und erweitern)

## Entwicklung

```bash
npm install
npm run dev
```

Die App läuft dann auf [http://localhost:3000](http://localhost:3000)
